Image Fetcher

This is an image fetcher that opens up API for searching for images on the web. You need to specify a query and then all images that match this query are returned as a JSON object.

Usage

There is only one API endpoint, /image/<query>. You can also specify the offset as a GET parameter, like so: /image/cat?offset=10
